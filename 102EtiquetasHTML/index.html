102 Etiquetas HTML

1 < !doctype >

Especifica el tipo de documento a crear 

                                
<!doctype html>

2 < html >

Representa la raíz del documento html 

                                
<html>
    .
    .
    .
</html>

3 < head >

Representa el encabezado de un documento html Aquí se coloca el titulo de la pestaña, se referencian las hojas de estilo y se coloca la metadata de la página. 

                                
<head>
    .
    .
    .
</head>

4 < body >

Representa el cuerpo de un documento html Aquí se coloca el contenido a visualizar de la página. 

                                
<body>
    .
    .
    .
<body>

5 < meta >

Es la etiqueta que define metadata para la página web 

                                
<meta charset="iso-8859-1">

6 < title >

Es la etiqueta que define el título de la pestaña de la página web 

                                
<title> 102 etiquetas html </title>

7 < link >

Este elemento es usado para incorporar estilos, códigos Javascript, imágenes o iconos desde archivos externos. 

                                
<head>
    <link rel="stylesheet" href="css/estiloFrame.css">
</head>

8 < header >

Este elemento provee información introductoria (títulos, subtítulos, logos) dentro del cuerpo del documento. 

                                
<!DOCTYPE html>
    <html>
        <head>
            <title>ejemplo</title>
            <link rel="stylesheet" href="css/estiloFrame.css">
        </head>
        <body>
            <header>
                Este es el título de la página
            </header>
        </body>
    </html>
